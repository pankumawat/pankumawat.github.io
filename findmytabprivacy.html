Privacy Policy

FindMyTabs

Last updated: April 10, 2026  ·  Effective date: April 10, 2026

FindMyTabs does not collect, transmit, store, or share any personal data — ever. Everything the extension does happens entirely inside your browser.

1. Who we are

FindMyTabs is a free, open-source Chrome extension developed and maintained by Pankaj Kumawat (github.com/pankumawat). The extension is published on the Chrome Web Store and its source code is publicly available on GitHub.

2. Data we do NOT collect

We do not collect, log, or transmit any of the following:

  • Tab titles or URLs you browse
  • Search queries typed into the extension popup
  • Custom rules or categories you create
  • Browsing history or session data
  • IP addresses, device identifiers, or location
  • Any personally identifiable information (PII)
  • Crash reports or usage analytics

3. Local storage only

The only data FindMyTabs ever writes to storage is your custom categorization rules (e.g. "tabs matching mycompany.com → Work"). This data is saved using chrome.storage.sync, which syncs across your own Chrome profile via your Google account — it is never sent to any server operated by FindMyTabs.

You can delete this data at any time by removing all custom rules from the extension's Rules page, or by uninstalling the extension.

4. Permissions explained

FindMyTabs requests only the Chrome permissions it strictly needs to function. Here is what each permission is used for:

Permission Why it's needed
tabs Read the list of open tabs (title, URL, window ID) so the extension can display and search them. Tab data never leaves your browser.
tabGroups Create and update Chrome native tab groups when you click "Group tabs".
sessions Read the list of recently closed tabs so you can reopen them with one click.
storage Save your custom rules to chrome.storage.sync so they persist across browser restarts and sync with your Chrome profile.

5. No external network requests

FindMyTabs makes zero network requests of its own. It does not contact any external server, analytics service, CDN, or third-party API. There is no backend. All processing — searching, categorizing, grouping — happens locally in your browser using the Chrome extension APIs.

6. Third-party services

FindMyTabs does not integrate with or embed any third-party analytics, advertising, tracking, or data-collection services (such as Google Analytics, Sentry, Mixpanel, etc.).

The extension is distributed through the Chrome Web Store, which is operated by Google. Your use of the Chrome Web Store is governed by Google's Privacy Policy.

7. Open source

FindMyTabs is fully open source. You can inspect every line of code at:

8. Children's privacy

FindMyTabs is a general-purpose productivity tool not directed at children. We do not knowingly collect any information from anyone, including children under 13.
